FoodShare

Food Redistribution Desktop App (Python Tkinter)

Overview

Connects donors, volunteers, and NGOs to redistribute surplus food to people and animals.

Tech

Tkinter • Pandas • NumPy • Matplotlib • MongoDB / JSON

Run
pip install pymongo matplotlib pandas numpy pillow
cd foodshare
python main.py
Features
Donors: post food, mark urgent, track status, chat
Volunteers: accept pickups, deliver, chat
Shared: notifications, analytics dashboard, NGO directory
Summary

Reduces food waste by enabling fast, local redistribution through a simple desktop app.
